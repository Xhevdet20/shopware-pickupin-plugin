Test Assignment for Shopware Developers
Task:
Create a Shopware 6.7 plugin named Pickup in Store.
Description:
The plugin should ensure that when a customer selects the Pickup in Store shipping method during checkout, the shipping address of the order is automatically set to the store’s address.
If the customer has entered a different shipping address, it must not be used once Pickup in Store is selected.
Instead, the store’s address should always overwrite the customer’s shipping address in this case.
Requirements:
The plugin must be fully compatible with Shopware 6.7.
The implementation should follow Shopware best practices (Events, Services, Dependency Injection, etc.).
All settings and configurations should be manageable through the native Shopware options (e.g., admin settings, standard configuration).
The source code must be clean, well-structured, and easy to understand.
Submission:
Please send us the completed plugin as a ZIP file.
We will install the plugin and review it together with you during the interview by testing it in a real workflow.
